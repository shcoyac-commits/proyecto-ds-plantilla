# Proyecto Data Science — Plantilla

## Descripción
Plantilla base para proyectos de análisis de datos.

## Estructura
data/raw/          → datos originales (no se suben a GitHub)
data/processed/    → datos limpios
data/external/     → datos de fuentes externas
notebooks/         → análisis exploratorios
src/               → funciones reutilizables
reports/           → resultados y gráficas

## Environment
conda activate analytics

## Stack
- Python 3.11
- pandas, numpy, matplotlib, seaborn, scikit-learn
